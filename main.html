description 目的

同志諸君。

このサイトは、国際政治学や地政学の視座から、外交や政治の流れを読み解こうとする者たちのためにある。 混迷を深める世界情勢の背後にある力学を、冷静に、しかし確かに掴み取ってほしい。

gesture 操作要綱

  • 表示モード切替: 右上のボタンで、全球俯瞰と平面地図の表示を切り替える。
  • 対象国の選択: サイドバーから国家を選択すると、視点が対象国へ移行する。
  • イベントの分析（マップ→一覧）:
    • マップ上のマーカー（線：二国間イベント、点：単一国家イベント）をクリックすると、下部データ一覧の該当行が明示される。
    • マーカーにカーソルを合わせる（ホバー）と、イベントの概要が簡易表示される。
  • イベントの特定（一覧→マップ）: 下部のデータ一覧から行を選択すると、マップ上の該当するイベントへ視点が移動する。
  • 凡例の活用: マップ右上の凡例で、イベント種別と色分けの対応を確認できる。凡例はクリックによる開閉が可能である。

warning 留意事項

  • 本サイトでは、GDELTプロジェクトから15分ごとに自動更新される最新データ（Last 15 Minutes CSV Data File List – GDELT Translingual.）を用いているが、その情報の完全性や正確性を保証するものではない。
  • 必ず「参照」列にある「記事を読む」リンクから元のニュースソースを確認し、その信頼性を諸君自身の目で判断してほしい。
