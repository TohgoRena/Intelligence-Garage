このサイトについて

description サイトの目的

このサイトは、世界情勢をリアルタイムで可視化することにより、自国や他国に興味を持つようになることを目的としています。

gesture 使い方

  • 表示切替: 右上の「3D Globe / 2D Map」ボタンで、表示モードを切り替えられます。
  • 国フォーカス: 左側のサイドバーから国名をクリックすると、その国に視点が移動します。
  • イベント詳細:
    • マップ上の線（2国間イベント）や点（単一国イベント）をクリックすると、下のテーブルの該当行がハイライトされます。
    • 線や点をホバーすると、イベントの概要が表示されます。
  • テーブル連携: テーブルの各行は、マップ上のイベントと連動しています。
  • 凡例: マップ右上の凡例で、イベントの種類と色の対応を確認できます。凡例はクリックで開閉できます。

warning 注意事項

  • GDELTプロジェクトから15分ごとに自動更新される最新のデータ（Last 15 Minutes CSV Data File List – GDELT Translingual.）を取得して使用していますが、情報の完全性や正確性を保証するものではありません。
  • イベントの場所や関係者は機械的に抽出されているため、誤りが含まれる可能性があります。この可視化は参考程度に留め、参照先を必ず読むようにしてください。
  • 詳細は、各イベントの「参照」列にある「記事を読む」リンクから元のニュースソースをご確認ください。参照先の記事が信用できるかどうかはご自身で判断してください。
